Sistema IC

Git + Jenkins + Docker + Pulp


Realizado para Samsung Zhilabs, consiste en un sistema de integración continua donde Git, Jenkins, Docker y Pulp trabajan mano a mano para cerrar el circuito desde que se añade una nueva línea de código hasta que se distribuye el producto al cliente. Git actúa como el repositorio donde se encuentra el código, Jenkins como el cerebro encargado de gestionar y ejecutar los distintos procesos, Docker como el entorno de ejecución de dichos procesos y, por último, Pulp como repositorio donde se sube y distribuye el producto resultante.
